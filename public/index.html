Image Search microservice

API Basejump: Image Search Abstraction Layer

Description:

This microservice gives you the ability to serach for images on the web via a custom search engine from google. After /api/imagesearch/ simply add the terms you want to serach for (with %25 as a seporator) and end with the query ?offset="integer" which will allow you to paginate through the result. The result will give you an array of 10 objects and each object will have:

  • the url to an image matching the serach term
  • a snippet of a paragraph from the origninal page
  • a link to to thumbnail of hte image in question
  • a link to the original site whre the image was published

You can also look through the most recent searches done by any user if you type in /api/latest/imageserach. The result includes an array of ten objects, each contaning the term the user searched for plus the timestamp of when the search was made.
