Image Search microservice

API Basejump: Image Search Abstraction Layer

Description:

This microservice gives you the ability to serach for images on the web via a custom search engine from google. After /api/imagesearch/ simply add the terms you want to serach for (with %25 as a seporator) and end with the query ?offset="integer" which will allow you to paginate through the result. The result will give you an array of 10 objects and each object will have:

  • The link to an image matching the serach terms
  • A snippet of the alternate text from the image
  • A link to a thumbnail of the image in question
  • A link to the original site from where the image was uploaded

You can also look through the most recent searches done by any user if you type in /api/latest/imageserach. Here you will se an array of ten objects, each contaning the term the user searched for plus the timestamp of when the search was made.
