Image Search microservice

API Basejump: Image Search Abstraction Layer

Description:

This microservice will enable you to search for images on the web via a custom search engine from google. Locate to /api/imagesearch/, then simply add the terms you want to serach for (with %25 as a seporator) and end the address with a query ?offset="integer". This "integer" will allow you to paginate through the results; each page will show you an array of ten objects and each object will have:

  • The link to an image matching the search terms
  • A snippet of the alternate text from that image
  • A link to a thumbnail of the image in question
  • A link to the original page from where the image was uploaded

You can also look through a list of the ten most recent searches if you type in /api/latest/imagesearch. Here you will see an array of objects, each contaning the term(s) the user searched for plus the timestamp (in GMT) of when the search was made.
